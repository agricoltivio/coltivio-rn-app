# Welcome to Pastorino React Native app 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app). Though, expo-router was ejected from this project and follows a more traditional "React Navigation" app. The entry point is in the root, App.tsx.

## Get started

1. Install dependencies

   ```bash
   yarn install
   ```

2. Start the app

   ```bash
    yarn ios | yarn android
   ```
